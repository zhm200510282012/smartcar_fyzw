# 最终注意事项

## P0

- P2.2 到 ESC PWMIN 和 GND 共地必须实测。
- 首次风机测试必须离墙、低脉宽、可立即断电。
- 不确认风机吸力、ESC 响应、IMU 轴向前，禁止真实上墙。
- `FAN_ESC_PHYSICAL_OUTPUT_ENABLE=0`、`WALL_RUN_ENABLE=0`、`SUCTION_HW_VERIFIED=0` 是当前安全默认。

## P1

- U9 的 A-E 到 ADC 物理顺序必须实测。
- 左右电机正方向必须实测。
- 左右编码器接口与 A/B 相位必须实测。
- 轮径、轮距、编码器比例必须实测。
- IMU pitch 正负与安装轴向必须实测。

## P2

- 上墙前距离、下墙前距离、终点距离需要赛道实测后写入 `competition_profile.h`。
- 不同赛段速度需要从低速开始逐步提高。
- 电磁阈值和丢线阈值需要用真实赛道数据修正。
- 风机 PRECHARGE/HOLD/BOOST 脉宽只能在台架确认后调整。

## 不得误判

- Host-SIL 全部通过不等于真实车能上墙。
- Keil 编译通过不等于电机、风机、IMU 方向正确。
- HEX 存在不等于可以立即烧录上墙。
- 首次完整赛道必须低速、低风险、可断电。
