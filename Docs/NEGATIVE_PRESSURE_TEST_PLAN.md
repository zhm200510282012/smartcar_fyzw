# 负压测试计划

| 阶段 | 测试内容 | 通过判据 | 失败回退 |
|---|---|---|---|
| A. 电气与信号确认 | 主电断开，示波 P23；确认 50 Hz、OFF、1 ms、2 ms、无毛刺 | 波形与 ESC 手册一致 | `SUCTION_HW_VERIFIED=0` |
| B. 固定工装 ESC/风机确认 | 连接 ESC PWMIN/GND，风机固定，低风险启停 | 方向正确，小于 1 ms 不转，1-2 ms 可控，大于 2 ms 停转 | 禁止上墙 |
| C. 地面预充与斜坡 | 低速/车固定验证 PRECHARGE 斜坡 | 推进限速，负压命令平滑 | 回 OFF |
| D. 安全绳静态贴壁 | 推进禁能，验证静态附着 | 可人工确认余量，温升可控 | 不进入动态 |
| E. 低速地面-圆弧-墙面 | 最低速度上 20 cm 以上过渡圆弧 | 连续姿态确认，无丢线 | 保压等待 |
| F. 垂直面稳态循迹 | 低速线控，记录风险和温升 | 可重复通过 | 降速/禁用 |
| G. 圆筒/跷跷板 | 单元素分别测试 | 单项通过后再全程 | 禁用元素 |
| H. 连续全程与热管理 | 多次运行，记录电池和 ESC 状态灯 | 无保护触发 | 降额或停止 |
