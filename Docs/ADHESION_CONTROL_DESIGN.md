# 负压调度与附着风险设计

## 协议事实

ESC 手册说明控制口 PWM 输入频率为 50-300 Hz，通过高电平时间控制速度；1-2 ms 为工作区间，小于 1 ms 不转，大于 2 ms 会认为无输入并停转，推荐 50 Hz。

官方 `BLmotor_Init_1(50)` 使用 P23 单路无刷候选输出，`BLmotor_Ctrl_w1(1000)` 注释为 50 Hz 时 1 ms。

## 当前禁能策略

- `SUCTION_HW_VERIFIED=0`。
- 所有 `SUCTION_*_NATIVE` 默认 0。
- `bsp_suction_apply()` 在编译期分支中强制 `SUCTION_OFF`。
- 只有完成示波、接线、供电、方向、固定工装后，才能把安全值改为实测微秒脉宽。

## 风险估计

`adhesion_risk` 使用表面状态、IMU 新鲜度、线质量、编码器健康、电源健康、命令持续性估计风险。它不是负压传感器读数，也不能证明吸附成功。

风险上升动作顺序：限速 -> 稳定转向 -> 请求 BOOST -> 停车保压等待人工。
