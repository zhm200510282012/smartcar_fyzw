# Full Course Release Criteria

Software release gates are separate from real-car release gates.

| Gate | Schematic confirmed | Code implemented | Host-SIL verified | Real-car pending |
| --- | ---: | ---: | ---: | ---: |
| Boot to Arm to Ground Track with valid sensors | no | yes | RUNTIME01/RUNTIME02 | yes |
| Five-channel line chain | no | yes | RUNTIME03 | yes |
| Differential speed loop unit separation | no | yes | RUNTIME06 | yes |
| Wall precharge flow | software only | yes | RUNTIME08-RUNTIME10 | yes |
| Fan failsafe hold behavior | P2.2 mapped only | yes | RUNTIME11/RUNTIME12 | yes |
| Full-course segment telemetry | software only | yes | final full-course Host-SIL | yes |
| P0 safety gates | software only | yes | P0 13/13 | yes |
| Keil C251 build | no | project updated | blocked this run | yes, run on a machine with Keil available |

Current hard locks:

```c
#define FAN_ESC_PHYSICAL_OUTPUT_ENABLE 0
#define WALL_RUN_ENABLE 0
#define SUCTION_HW_VERIFIED 0
#define ROUTE_PROGRESS_SCRIPT_ENABLE 0
```

Release requirements before wall testing:

1. Keil C251 rebuild of `AI8051U_FYZW_SAFE` with 0 errors and 0 warnings.
2. Fresh HEX generated by that rebuild.
3. Power ADC calibration and low-voltage threshold set with `POWER_GUARD_ENABLE=1`.
4. Five electromagnetic channels confirmed for physical order, offset, gain, and invert.
5. LSM6DSR SPI and pitch axis confirmed.
6. Encoder counts per wheel revolution and wheel circumference measured.
7. P2.2 ESC signal bench verified, with common ground and safe arming behavior.
8. `WALL_RUN_ENABLE` and `FAN_ESC_PHYSICAL_OUTPUT_ENABLE` remain 0 until the above gates pass.

Host-SIL and Keil passing do not prove real track completion.
