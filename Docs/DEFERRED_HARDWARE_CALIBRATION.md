# 待实车确认项目

本轮软件架构已接通，但以下项目不能由代码或 Host-SIL 代替实物确认。

| 项目 | 当前处理 | 不确认的风险 | 最小确认动作 |
| --- | --- | --- | --- |
| U9 A-E 到 ADC 线序 | 映射集中在 `BSP/board_emag_map.h`，算法只认逻辑 A-E | 左右误差反向、丢线误判 | 单独遮挡/靠近每个电感，记录 ADC 通道和 A-E 顺序 |
| 左右编码器接口与正方向 | 左右 PI 独立，invalid/stale 归零 | PI 反向、速度估计错误 | 离地低速转左/右轮，确认计数正负和新鲜度 |
| 轮径、轮距、编码器比例 | 默认保守速度，不宣称真实 mm/s 精确 | 速度闭环比例错误，差速转向半径错误 | 推车或低速直线测距，修正比例参数 |
| IMU 轴向 | `IMU_PITCH_SIGN`、`IMU_PITCH_OFFSET_CDEG` 可配置 | 上墙/下墙误判或反向 | 静止地面、抬头、墙面姿态三点记录 pitch |
| 风机 ESC 实物控制线与共地 | `BOARD_FAN_PWM_MAPPED=1`，物理输出仍禁用 | 无响应、误启动、信号悬空 | 主电断开示波 P2.2/PWMIN/GND，确认共地 |
| 风机实际起转/保持/增强脉宽 | `FAN_*_US` 仅为软件起点 | 起转失败、过流、吸附不足 | 固定工装中从 1000 us 起逐步测试，记录起转、保持、boost |

完成这些确认前：

```text
FAN_ESC_PHYSICAL_OUTPUT_ENABLE 必须保持 0；
WALL_RUN_ENABLE 必须保持 0；
不允许上墙；
Host-SIL 和 Keil 通过不等于实车通过。
```
